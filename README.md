# Description 

Bot discord with breakpoints slashs commands with API

# Methods : 

GET, POST, PUT, DELETE

# Slashs Commands: 

- /festival-create
- /festival-update-by-id
- /festival-get-all
- /festival-get-by-id
- /festival-destroy-by-id

# COMMANDS

Install the node modules 

```console
    npm install
```

Then run the project with it

```console
    npm run dev
```

# Entity

Festival :

name: String,
startDate: Date,
endDate: Date,
numberPlaces: Integer
